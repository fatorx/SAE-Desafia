# Instalação

 - PHP 7.3
    1. PDO Extension
    2. SQLLite Extension
 - Executar o comando na raiz do projeto
    > php composer.phar install   
 - Iniciar o servidor do PHP pelo comando
    > php -S localhost:9098 -t public/
